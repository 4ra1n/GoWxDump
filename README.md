# GoWxDump
## 特别说明
GoWxDump是<a href="https://github.com/AdminTest0/SharpWxDump">SharpWxDump</a>的Go语言版本。
## 使用方法
### 1.下载
```
git clone https://github.com/SpenserCai/GoWxDump.git
```
### 2.编译
```
go mod tidy
set GOARCH=386
go build
```
### 3.使用
```
GoWxDump.exe
```
